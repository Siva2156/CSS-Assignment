HTML Special Characters

Comments and Special Characters

In HTML, certain characters have special meanings and need to be represented using HTML entities to display correctly.

The ampersand symbol (&) should be written as &.

The less-than symbol (<) should be written as <.

The greater-than symbol (>) should be written as >.

The copyright symbol (©) should be written as ©.

The registered trademark symbol (®) should be written as ®.

The trademark symbol (™) should be written as ™.




  • Ampersand: &
  • Less-than: <
  • Greater-than: >
  • Copyright: ©
  • Registered Trademark: ®
  • Trademark: ™
